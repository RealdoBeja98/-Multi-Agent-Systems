┌─────────────────────────────────────────────────────────────┐
│  KEY CONCEPTS                                               │
└─────────────────────────────────────────────────────────────┘

  Agents:
    • coordinator    → Manages task allocation
    • robot1-4       → Autonomous transport robots
    • sawing         → Production station
    • drilling       → Production station

  Process:
    1. Station needs materials → Sends REQUEST
    2. Coordinator → Broadcasts CFP to robots
    3. Robots → Calculate cost and PROPOSE bid
    4. Coordinator → Selects lowest bid, assigns task
    5. Winner → Executes transport
    6. System → Continues autonomously

  Technology:
    • Prometheus methodology (3 phases)
    • JADE platform (FIPA-compliant)
    • Contract Net Protocol (bidding)
    • BDI architecture (intelligent agents)
