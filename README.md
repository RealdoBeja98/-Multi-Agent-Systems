# -Multi-Agent-Systems

To run this projcet you need to download jade 

Command to run on Powershell : 

Invoke-WebRequest -Uri "https://jade.tilab.com/dl.php?file=JADE-bin-4.6.0.zip" -OutFile "JADE-bin-4.6.0.zip"

Place the jar jade inside the directory warehouse-demo where other java files are. 

Inside that directory run : 

.\COMPILE.bat   

Other command : 

 .\RUN.bat
